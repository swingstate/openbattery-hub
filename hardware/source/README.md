# Source (editable design files)

- `ProPrj_OpenBattery Hub 1.9.3_2026-07-22.epro2` — the complete EasyEDA
  Pro project export (schematic, PCB layout, routing, footprints,
  symbols, board configuration). This is the "Complete Source" as
  defined in CERN-OHL-S v2 section 1.9 — open it in
  [EasyEDA Pro](https://easyeda.com/) (Import Project) to view or modify
  the schematic and PCB layout, including component placement and
  routing.
- `Netlist_Schematics_*.enet` / `Netlist_PCB_*.enet` — plain-text netlist
  exports (components, footprints, supplier/manufacturer part
  references, connectivity) for reference and tooling; derivable from
  the project file above.
